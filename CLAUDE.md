# רגע קוסמי — landing page

## Git workflow

After every change you make to any file in this project (index.html, copy files, images, etc.), automatically run:

```
git add .
git commit -m "<short description of the change>"
git push
```

Do this without asking for confirmation each time, unless the push fails
(e.g. due to a conflict or authentication issue), in which case stop and
explain the issue before proceeding.
